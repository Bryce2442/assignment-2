The Start

Transmission Control Protocol

The transmission control protocol was the internet before the Web we have in modern day. This was called the pre era. The pre era saw the development of various applications like emails, telnet, and early forms of streaming and gaming.

Creation of the web

The web was created in 1989 by Tim Berners-Lee. Lee introduced key technologies like HTML, URL, and HTTP. The name of the first browser was the WorldWideWeb. By the mid-1990s, the World Wide Web transitioned into the dynamic web.

Dynamic Web

The dynamic web brought interactivity and user-generated content. Technologies like JavaScript, AJAX, and database integration enabled websites to update content without refreshing the page. This paved the way for modern applications like social media, online banking, and streaming platforms.
